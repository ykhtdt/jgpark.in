# Personal Website

개인 웹사이트 및 이력서를 위한 모노레포 프로젝트입니다.

**(현재 이전 중입니다)**

## 프로젝트 구조

```
jgpark.in/
├── apps/
│   ├── resume/             # 이력서
│   └── web/                # 개인 웹사이트
├── packages/
│   ├── core/               # 핵심 유틸리티 및 컴포넌트 등
│   ├── eslint-config/
│   ├── typescript-config/
│   └── ui/
```

## 기술 스택

- React 19, Next.js 15
- TailwindCSS 4, Typescript
- PNPM, Turborepo
